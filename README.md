# EE Personal Tutor Bot 🚀

یک ربات تلگرام هوشمند برای دانشجویان مهندسی برق (EE) که از قابلیت‌های پیشرفته هوش مصنوعی برای آموزش، حل مسئله و مدیریت منابع درسی استفاده می‌کند.

## قابلیت‌های کلیدی

- **انتخاب درس**: پشتیبانی از دروس تخصصی برق (سیگنال، مدار، مغناطیس و ...).
- **سیستم RAG**: بارگذاری جزوات (PDF, DOCX) و جستجو در متن آن‌ها برای پاسخگویی دقیق.
- **تحلیل تصویر**: تشخیص قطعات مدار و تحلیل توپولوژی از روی عکس.
- **حل‌کننده هوشمند**: تشخیص خودکار سوالات محاسباتی و ارائه راه‌حل گام‌به‌گام.
- **ورودی صوتی**: تبدیل صدا به متن و پاسخگویی به سوالات شفاهی.
- **پیشرفت تحصیلی**: رهگیری سوالات و نتایج کوییزها.

## پیش‌نیازها

- **Python 3.12.10**
- یک ربات تلگرام (ساخته شده در `@BotFather`)
- کلید API از **LLM7.io** یا هر سرویس سازگار با OpenAI.

## نصب و راه‌اندازی

1. **کلون کردن پروژه**:
   ```bash
   git clone <repo_url>
   cd ee-tutor-bot
   ```

2. **نصب وابستگی‌ها**:
   ```bash
   pip install -r requirements.txt
   ```

3. **تنظیم تنظیمات**:
   فایل `.env.example` را به `.env` تغییر نام دهید و مقادیر مربوطه را پر کنید:
   - `TELEGRAM_BOT_TOKEN`
   - `LLM_API_KEY`

4. **اجرای ربات**:
   ```bash
   python main.py
   ```

## ساختار پروژه

- `main.py`: نقطه ورود برنامه.
- `src/bot/`: هندلرهای تلگرام و منطق منوها.
- `src/tutor/`: پرامپت‌های مهندسی شده برای مدهای مختلف (Solver, Circuit, QA).
- `src/retrieval/`: مدیریت پایگاه داده برداری (FAISS) و استخراج متن از فایل‌ها.
- `src/database/`: مدیریت کاربران و پیشرفت تحصیلی با SQLite.
- `src/llm/`: کلاینت ارتباط با مدل‌های زبانی.

## نحوه استفاده در تلگرام

- **/start**: شروع کار و نمایش منوی اصلی.
- **ارسال فایل**: جزوات درسی خود را بفرستید تا ربات آن‌ها را یاد بگیرد.
- **ارسال عکس**: از مدار یا فرمول خود عکس بگیرید تا ربات آن را تحلیل کند.
- **انتخاب درس**: از منوی "انتخاب درس" برای فوکوس کردن ربات روی یک موضوع خاص استفاده کنید.
